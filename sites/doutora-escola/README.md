# Doutora Escola — Landing Page

Site **independente** da página de vendas (Campos de Experiências / BNCC).  
Para ser publicado em **outro domínio**.

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse: `http://localhost:5174`

## Build para produção

```bash
npm run build
```

Saída em `dist/`. Sirva esse diretório no domínio desejado (Nginx, Apache, Vercel, etc.).

## Estrutura

- Página única (sem React Router).
- Componentes em `src/components/`.
- Imagens em `public/images/doutora-escola/`.
