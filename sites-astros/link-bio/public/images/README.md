# Imagens

Coloque aqui a **foto (avatar)** para aparecer no topo do link na bio:

- **Nome do arquivo:** use **um** destes (o Header tenta na ordem): `avatar.webp`, `avatar.jpg` ou `avatar.png`
- **Caminho completo no projeto:** `sites-astros/link-bio/public/images/avatar.webp` (ou .jpg / .png)

Exemplo: se a sua foto for `foto.jpg`, renomeie para `avatar.jpg` e coloque dentro da pasta `public/images/`.

Depois de colocar a foto, rode o build de novo: `./build-site.sh link-bio` (na pasta nginx-multisite).
