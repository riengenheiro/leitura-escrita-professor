# Imagens do site V2.0 – Coleção Campos de Experiências

Todas as imagens devem estar em **formato WebP** (.webp), proporção **1024 × 1536** px. Coloque os arquivos nesta pasta:

| Arquivo | Uso |
|---------|-----|
| `01-hero-mockup.webp` | Mockup do ebook V2.0 no hero (tablet/computador com o ebook aberto). Capa moderna, visual clean. |
| `bloco-1.webp` | Capa do Bloco 1 – O Eu, o Outro e o Nós |
| `bloco-2.webp` | Capa do Bloco 2 – Corpo, Gestos e Movimentos |
| `bloco-3.webp` | Capa do Bloco 3 – Traços, Sons, Cores e Formas |
| `bloco-4.webp` | Capa do Bloco 4 – Escuta, Fala, Pensamento e Imaginação |
| `bloco-5.webp` | Capa do Bloco 5 – Espaços, Tempos, Quantidades |
| `guaciara-foto.webp` | Foto profissional da Dra. Guaciara Fornaciari (seção “Quem criou”) |

Recomendações do briefing: cores educativas (azul, verde, laranja suave), imagens profissionais e clean.
