# Imagens da Fábrica Mágica

As imagens **precisam estar nesta pasta** para aparecer no site após o build.

| Arquivo         | Uso                    |
|-----------------|------------------------|
| `hero-fm.webp`  | Hero (mockup)          |
| `guaciara.webp` | Seção "Quem é Guaciara"|
| `og-image.webp` | Preview redes sociais  |

Build: `npm run build` — a pasta `dist/images/fabrica-magica/` conterá estes arquivos.
