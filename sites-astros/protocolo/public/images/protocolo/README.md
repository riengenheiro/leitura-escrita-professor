## 🎨 Recomendações

- **Formato:** WebP (obrigatório)
- **Qualidade:** Boa resolução, mas otimizada (não muito pesada)
- **Estilo:** Fotografias naturais, não stock photos genéricas
- **Cores:** Harmonizar com a paleta do site (verde/lima)

---

## 📝 Após Adicionar as Imagens

1. Coloque os arquivos na pasta `public/images/protocolo/`
2. Faça o build:
   ```bash
   cd /var/www/projetos/nginx-multisite
   ./build-site.sh protocolo-vivasuamissap
   ```

As imagens estarão disponíveis em:
- `https://protocolo.vivasuamissao.top/images/protocolo/hero.webp`
- `https://protocolo.vivasuamissao.top/images/protocolo/testimonial-maria.webp`
- etc.
